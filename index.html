Vim Modes

Vim has Three basic modes of operation.

  • <ESC> - Normal Mode. Lets you navigate the file. The "Gateway Mode".
  • i - Insert Mode. Lets you edit the text.
  • v - Visual Mode. Lets you select text.

Vim Motions

  • h - Move Left one character
  • j - Move Down one character
  • k - Move Up one character
  • l - Move Right one character

Vim Motions

  • w - Move Forwards one Word
  • b - Move Backwards one Word
  • e - Move to End of next Word

Vim Motions

  • W - Move Forwards one Big Word
  • B - Move Backwards one Big Word
  • E - Move to End of next Big Word

What is a Big Word
Well... Big words are any non-whitespace character.

Vim Motions

  • ) - Move to Forward one Sentence
  • ( - Move to Backward one Sentence
  • } - Move Forward one Paragraph
  • { - Move Backward one Paragraph

Vim Motions

Remember: Any motion can be preceded by a number to let Vim know how many times you would like to repeat the motion.


For example:

  • 3w - Move 3 words forward.
  • 2{ - Move 2 paragraphs backward.
  • 900e - Move to the end of 900 words.

Vim Motions

You can also use the `.` command to rerun the last edit.

Vim Operators

  • c - Change Text
  • d - Delete Text
  • y - Yank Text

Vim Operators

Vim operators can not be used alone. They must be paired with a motion.
If you think about it, that really makes sense. You cannot just "change" or just "delete".
You can however "change a word" or "delete a word".

Vim Operators

  • cw - Change a word
  • dw - Delete a word
  • yw - Yank a word

Vim Operators

But wait... There is more than just those motions.

Vim Operators

  • c) - Change a sentence
  • y} - Yank a paragraph
  • d4w - Delete 4 words

Vim Text Objects

Vim Text Objects

Text objects bring even more power to your Vim editing.
Now instead of needing to move to the beginning of a word to perform an operation on it you can use a Text object.

  • i - Inside of
  • a - Around

Vim Text Objects

  • ciw - Change Inside of a Word.
  • caw - Change Around a Word.
  • diw - Delete Inside of a Word.
  • daw - Delete Around a Word.
  • yiw - Yank Inside of a Word.
  • yaw - Yank Around a Word.
