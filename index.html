Vim Modes

Vim has Three basic modes of operation.

  • <ESC> - Normal Mode. Lets you navigate the file. The "Gateway Mode".
  • i - Insert Mode. Lets you edit the text.
  • v - Visual Mode. Lets you select text.

Vim Motions

  • h - Move Left one character
  • j - Move Down one character
  • k - Move Up one character
  • l - Move Right one character

Vim Motions

  • w - Move Forwards one Word
  • b - Move Backwards one Word
  • e - Move to End of next Word

Vim Motions

  • W - Move Forwards one Big Word
  • B - Move Backwards one Big Word
  • E - Move to End of next Big Word

What is a Big Word
Well... Big words are any non-whitespace character.

Vim Motions

  • ) - Move to Forward one Sentence
  • ( - Move to Backward one Sentence
  • } - Move Forward one Paragraph
  • { - Move Backward one Paragraph

Vim Motions

Remember: Any motion can be preceded by a number to let Vim know how many times you would like to repeat the motion.


For example:

  • 3w - Move 3 words forward.
  • 2{ - Move 2 paragraphs backward.
  • 900e - Move to the end of 900 words.

Vim Motions

You can also use the `.` command to rerun the last edit.

Vim Operators

  • c - Change Text
  • d - Delete Text
  • y - Yank Text

Vim Operators

Vim operators can not be used alone. They must be paired with a motion.
If you think about it, that really makes sense. You cannot just "change" or just "delete".
You can however "change a word" or "delete a word".

Vim Operators

  • cw - Change a word
  • dw - Delete a word
  • yw - Yank a word

Vim Operators

But wait... There is more than just those motions.

Vim Operators

  • c) - Change a sentence
  • y} - Yank a paragraph
  • d4w - Delete 4 words

Vim Operators

Double up on an operator to perform the operation on a line.

  • cc - Change the current line
  • dd - Delete the current line
  • yy - Yank the current line

This is typically the case for most actions in Vim.

Vim Operators

Uppercase variants perform the operation from your where your cursor is to end of the line

  • C - Change until end of current line
  • D - Delete until end of current line

This is typically the case for most actions in Vim.

Note, The exception is Y which yanks the current line.

Vim Text Objects

Vim Text Objects

Text objects bring even more power to your Vim editing.
Now instead of needing to move to the beginning of a word to perform an operation on it you can use a Text object.

  • i - Inside of
  • a - Around

Vim Text Objects

  • ciw - Change Inside of a Word.
  • caw - Change Around a Word.
  • diw - Delete Inside of a Word.
  • daw - Delete Around a Word.
  • yiw - Yank Inside of a Word.
  • yaw - Yank Around a Word.

Macros

Macros are a really great way for speeding up mundane tasks.

To start recording a macro press q and then a register to start recording to.

Macros

To stop recording the macro press q again.

Run macros by pressing @ and then the register that you stored the macro in.

Macros

  • qa - Start recording in register a
  • q - Stop the recording of a macro
  • @a - Playback the recording stored in register a

Macros

Remember that Vim is recording keystrokes and not actions. Make your macros repeatable by using more advanced motions.

Visual Mode

  • v - Enter visual mode in characterwise mode
  • V - Enter visual mode in linewise mode
  • c-v - Enter visual mode in blockwise mode
  • v - Exits visual mode when in visual mode

Visual Mode

Use Blockwise visual mode or c-v for performing the same task on multiple lines.

Use I to insert on every line on the leading edge of the Blockwise Visual selection

Use A to insert on every line on the trailing edge of the Blockwise Visual selection

Command mode

All of Vim's modes have commands associated with them.

  • :norm[al] - Perform action in Normal mode with command mode
  • :i[nsert] - Perform action in Insert mode with command mode
  • :vi[sual] - Perform action in Visual mode with command mode

Command mode

All of Vim's operators also have commands associated with them.

  • :c[hange] - Change text with command mode
  • :d[elete] - Delete text with command mode
  • :y[ank] - Yank text with command mode

Command mode

Why spend time learning these if we can just use the keys we learned before?

Command mode

Super charge macros and visual mode with :g and :v

Command mode

  • :g[lobal] - Perform an action on lines matching a regex
  • :g[lobal]! - Perform an action on lines not matching a regex
  • :v[global] - Same as :g!. Think of the inverse of global

Command mode

So how do I use :global??

:global/<regex_pattern>/<command_to_perform>

  • :global/^Remove me$/delete
  • :global/yank \w\+ me/yank

Command mode

But wait there is more!

  • :g/Some regex/normal @a

Command mode

But wait there is even more!!!

You can perform these actions over a specific range in your file

  1. Visually select a region in your file
  2. :g/run macro/normal @q

Command mode

Notice how Vim automatically inserts :'<,'> before g.

Vim is telling this command what range to perform the command over. By default the :global command runs over the entire file.

Most commands accept a range that you can use to perform that command over an area in a file.

Command mode

Ranges come before a command and tell that command where to run.

  • :42 - Perform command on Line 42
  • :1 - Perform command on first line
  • :$ - Perform command on last line
  • :. - Perform command on current line
  • :% - Perform command on all lines
  • :+3 - Perform command on next 3 lines

Command mode

Ranges come before a command and tell that command where to run.

  • :10,20 - Perform command on lines 10 to 20 (Inclusive)
  • :21,$ - Perform command on lines 21 to end of file
  • :70,+5 - Perform command on lines 70 to 75 (Inclusive)

Command mode

  • :%s/search/replace/ - Search and replace over entire file
  • :21,$g/search/normal @a - Run macro from line 21 to end of file
  • :70,+5delete - Delete lines 70 through 75 (Inclusive)

Command mode

Without a command Vim will move your cursor to that line

  • :10 - Move to line 10
